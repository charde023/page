```
① 회사 개요
   - 회사명 / 주생산품 / 주사업 / KSIC 코드
② 연구조직 개요
   - 연구소명 / 설립(신고) 목적
③ 전문 연구분야  ← 과학기술 분류로!
④ 연구개발 목표 (중단기)
⑤ 연구과제 목록  (과제명 / 연구기간 / 담당연구원)
⑥ 연구개발 내용 및 방법
⑦ 연구개발 필요성 (현 기술 한계·문제점, 정량 근거)
⑧ 기대효과 (기술적 / 경제적 / 사업화·지식재산)
```

## ✅ 에이폼 예시 A — 플라스틱 성형 분야

```
① 회사 개요
   주식회사 에이폼 / 포장용 플라스틱 성형용기·기타 플라스틱 제품 제조,
   전자상거래, 창고·물류업 / KSIC C2220 등
② 연구조직 개요
   ○○연구소 — 자사 제조·물류 공정의 기술적 난제를 자체 연구개발로 해결
③ 전문 연구분야 : 화학공학 / 고분자소재공학
④ 연구개발 목표
   식품안전 기준을 충족하면서 원재료 사용량 12% 절감하는 경량화 설계와,
   성형 온도·사출 속도 최적 구간을 과학적으로 도출한다.
⑤ 연구과제 목록
   - (과제1) 식품포장용 HDPE 블로우 성형 용기의 두께 균일성 및
     낙하충격 강도 향상을 위한 성형 공정조건 최적화
     / 2026.07~2027.06 / 담당: ○○○ 연구원
⑥ 연구개발 내용 및 방법
   - 원재료(HDPE·PP) 물성변수(MI·밀도) 조합 실험설계(DOE)
   - 성형온도(180~220℃)·블로우압력(4~8bar)·냉각시간 구간별
     시제품 30배치 제작 및 두께편차 측정
   - KS M 3100 기준 낙하충격·압축강도 시험
   - 최적조건 파라미터 도출 및 재현성 검증
⑦ 연구개발 필요성
   현재 HDPE 용기 불량률 4.2%, 두께편차로 인한 충격파손이 전체 클레임의
   38%. 성형 메커니즘이 과학적으로 규명되지 않아 경험적 조정에 의존 중 →
   체계적 연구 필요.
⑧ 기대효과
   - 기술적: 성형 불량률 1% 이하, 최적 공정 파라미터 DB 구축
   - 경제적: 원재료비 연 약 3,200만원 절감
   - 지식재산: 성형조건 최적화 노하우 특허 출원 검토
```

## ✅ 에이폼 예시 B — 스마트 물류 분야

```
③ 전문 연구분야 : 컴퓨터·소프트웨어 / 산업공학
④ 연구개발 목표
   자사 D2C 플랫폼 일 주문데이터(SKU 약 800종)를 학습데이터로,
   수요예측 오차율(MAPE) 15% 이하, 피킹 이동거리 20% 단축을 달성하는
   WMS 내장 알고리즘을 자체 개발한다.
⑤ 연구과제 목록
   - (과제2) 전자상거래 주문데이터 기반 AI 수요예측 알고리즘 및
     피킹 동선 최적화 WMS 모듈 개발 / 2026.07~2027.06 / 담당: ○○○
⑥ 연구개발 내용 및 방법
   - 2년치 주문이력 전처리, 계절성·프로모션 변수 추출
   - ARIMA·Prophet·LSTM 비교실험 및 앙상블 설계
   - 창고 레이아웃 기반 그래프탐색(TSP 변형) 알고리즘 구현
   - WMS 프로토타입 개발 → 실창고 파일럿 4주+ 테스트
⑦ 연구개발 필요성
   재고 발주가 담당자 경험에 의존, 과잉재고·품절 손실 월평균 약 280만원.
   범용 WMS의 수요예측 모듈이 자사 특성(이커머스 계절성·소용량 다품종)에
   부적합 → 자체 알고리즘 연구 필요.
⑧ 기대효과
   - 기술적: 수요예측 정확도 향상, 재고회전율 1.4→1.8배
   - 경제적: 재고손실 연 약 3,360만원 절감 추정
   - 지식재산: 물류 최적화 알고리즘 SW 저작권 등록
```
