부지·공장 면적이 아니라, 아래 4가지가 인정의 핵심입니다.

기업 규모별 최소 인원 충족. 에이폼은 창업 3년 이내라 2명 기준 적용 가능.

충족 가능

② 독립 연구공간

생산·영업·일반사무와 분리된 전용 공간. 사무동 2층 활용 시 충족 용이.

확인 필요

③ 연구 전용 기자재

연구개발 전용 PC·시험·측정·시제품 장비. 생산·품질 공용장비는 제외.

연결 용이

④ 실제 연구개발활동

제조·물류·커머스 등 매출사업과 직접 연결되는 연구주제 + 증빙체계.

  • 벤처기업 인증을 받으면 창업 3년 시한 부담을 없앨 수 있음

3연구전담요원 자격 조건

학력·전공

원칙적으로 자연계 분야 학사 이상. 산업디자인·서비스 분야는 비자연계 전공도 인정 가능. 중소기업은 전문대·마이스터고·특성화고 졸업 + 연구경력으로도 가능성.

주의

겸직 금지

연구업무 외 생산·판매·영업·물류·일반관리 겸직 불가. 연구 외 업무를 수행할 별도 상시종업원을 회사가 보유해야 함.

4독립 연구공간 조건

에이폼 시설(부지 1,000평 · 공장 400평 · 사무동 33평 2층)에 맞춘 안전한 구성.

권장

사무동 2층 = 연구소 본공간

연구전담요원 좌석, 전용 PC, 자료 보관함, 연구소 현판. 독립성 강하고 사후관리 리스크 낮음.

공장 일부 = 시험·시제품 구역

시제품 제작대·시험·측정 장비를 생산라인과 별도 구획(벽체·출입문 또는 명확한 구분).

  • 사무공간 일부를 연구소로 쓰려면 고정 벽체 + 별도 출입문이 원칙 (간이 파티션은 불인정 경향)
  • 단, 전용면적 50㎡ 이하면 일정 분리구역 방식도 예외적으로 가능
  • 회의실 형태만으로는 불인정 가능 → 좌석·PC·기자재·문서보관 공간이 실제로 있어야 함
  • 생산라인·물류창고·일반사무 공간은 연구소 면적에서 제외하는 것이 안전

  • 연구기자재는 연구개발용으로만 사용, 연구소 내부 또는 독립공간에 비치
  • 타 부서 공동사용·생산시설 병행사용 장비는 연구기자재로 인정받기 어려움

6연구개발활동·증빙 조건

연구주제는 사업자등록 업종 + 보유 시설과 연결되어야 실체 입증이 쉽습니다.

★ 권장

제조(플라스틱) 분야

포장용기 시제품 설계·성형조건 최적화, 소재·금형·공정 개선. → 제조업 업종과 직결.

★ 권장

물류·커머스 분야

창고 피킹 동선 최적화, 수요예측·자동발주, WMS/OMS, 냉장물류 온도관리. → 창고업·전자상거래와 직결.

  • 증빙: 연구개발계획서, 연구노트, 회의록, 결과보고서를 과제별로 작성·보관
  • 인건비·재료비·장비비·외주개발비를 구분관리해야 세액공제 방어 가능
