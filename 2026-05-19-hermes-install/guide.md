## 결론 (TL;DR)

> **윈도우는 이제 WSL 없이 PowerShell(관리자 권한)에 명령어 한 줄 붙여넣으면 끝. 설정 마법사는 거의 다 기본값(엔터), 프로바이더만 무료 Nous Portal(딥시크)로 고르면 된다. 마지막에 `hermes` 입력 → 노란 화면이 뜨면 성공.**

### 헤르메스 한 줄 정의

서버에서 24시간 돌아가는 **자율 자가발전 에이전트**. 작업하며 학습 → 해결 방식을 스킬로 저장 → 다음엔 더 잘함. 쓸수록 나에게 맞춰진다.

### 설정 마법사 — 선택값 한눈에

| 항목 | 선택 | 이유 |
|---|---|---|
| 오픈클로 설정 임포트? | **n** | 설정 꼬임 방지 |
| 퀵셋업 / 풀셋업 | **1 퀵셋업** | 오늘은 간단히 |
| 프로바이더 | **1 Nous Portal** | 무료 티어 (헤르메스 제작사) |
| 모델 | **1 딥시크 V4 플래시** | 무료·무제한, 잡일용 토큰 절약 |
| 터미널 백엔드 | **1 로컬** | 내 컴퓨터에 설치 |
| Gateway Working Dir | **엔터** | 기본 경로 |
| Enable sudo support | **N (무조건)** | 관리자 권한 — 위험, 처음엔 끄기 |
| 툴 프로그레스 | **All / 엔터** | |
| 컴프레션 | **0.5 / 엔터** | 기본값 |
| 세션 리셋 모드 | **1 디폴트** | 24h 구동·새벽 4시 리셋 |
| 인액티비티 타임아웃 | **1440분 / 엔터** | =24시간 |

> 잘못 골라도 OK — `hermes setup`으로 언제든 처음부터 재설정 가능.

## 강의는 어떤 내용이었나

지피터스 파트너 강성준 강사가, 22기 본 스터디 시작 전 모두를 위한 **헤르메스 설치 공통 세션**을 진행했습니다. 헤르메스가 무엇인지 짧게 짚고, 윈도우·맥을 양방향으로 보며 설치를 끝까지 따라가는 워크스루입니다. 강사는 맥에 **VMWare Fusion**으로 윈도우 가상 머신을 띄워(보안 격리) 윈도우 위주로 진행했습니다.

| 섹션 | 내용 |
|---|---|
| 개념 | 헤르메스 = 24h 자가발전 에이전트, 도구 비교, 카파시 "병목론" |
| 환경 | 맥 + VMWare 가상 윈도우 (보안 격리) |
| 실습 | 명령어 복사 → PowerShell 관리자 실행 → 설정 마법사 → 실행 |
| 마무리 | (시간 부족) 텔레그램 연동·활용은 본 스터디에서 |

## 0. 한눈에 보는 설치 흐름

1. Nous Research 설치 페이지에서 OS별 명령어 복사
2. PowerShell(관리자 권한) / 맥 터미널 열기
3. 명령어 붙여넣기 → uv가 파이썬 등 자동 설치
4. 설정 마법사 진행 (위 표대로)
5. Nous Portal 무료($0) 로그인·연결
6. `hermes` 입력 → 노란 화면 = 설치 성공

## 1. 헤르메스란 & 왜 쓰나

- **24시간 자가발전 에이전트**: 세팅하면 알아서 돌고, 스킬을 스스로 만들고 성장. 쓸수록 맞춤형이 되고 성능이 올라감.
- **원리**: 작업 수행 → 학습 → 해결 방식을 스킬로 저장 → 다음에 더 나은 결과 → 반복.

> 안드레 카파시: **"내가 병목이 돼서는 안 된다."** 사람이 프롬프트 사이에 껴 있으면 에이전트가 원활히 못 돈다. 헤르메스 같은 에이전트를 잘 쓰면 사람이 루프 밖으로 나와 **슈퍼바이저(관리)** 역할만 하게 된다.

역할 분담 예시: 코드 작업은 **클로드 코드**, 딥리서치·모니터링·자동화·일정·이메일·LLM 위키는 **헤르메스**. 단 컨텍스트(맥락)는 서로 공유되게.

---

## 2. 실습 — 설치 단계별

### Step 1. 명령어 복사

Nous Research 공식 설치 페이지에서:
- **맥**: `curl`로 시작하는 위쪽 명령어
- **윈도우**: 아래쪽 명령어 (이제 **WSL 없이 PowerShell 네이티브** 설치 가능 — 베타라 약간 깨질 수 있으나 대체로 OK)

### Step 2. 터미널 열기

- **윈도우**: 시작 → "파워쉘" 검색 → **오른쪽 클릭 → 관리자 권한으로 실행** → "디바이스 변경 허용" 예
- **맥**: 스팟라이트에서 "터미널" 검색 → 실행

### Step 3. 붙여넣기 → 설치

커서 위치에 명령어 붙여넣고 엔터. **uv 패키지 매니저**가 파이썬 등 필요한 것을 자동 설치(처음이면 시간이 더 걸림).

### Step 4. 설정 마법사

위 TL;DR 표대로 진행. 핵심 포인트:

- **프로바이더 vs 모델**: 프로바이더=회사(오픈AI·엔트로픽·구글), 모델=코덱스·클로드·제미나이.
- **무료로 시작**: Nous Portal(1) → 로그인 → 구글 로그인 → **Free($0)** → **Subscribe and Connect**. 0원이지만 카드 정보는 요구(실제 결제 X). 헤르메스에 잡일을 많이 시키니 무료 딥시크가 토큰 절약에 유리.
- **구독자**: 코덱스 구독자는 코덱스 선택(GPT 최상위), 클로드 코드 구독자는 엔트로픽. 단 **클로드는 API라 비용 큼 → 헤르메스에선 비추천.**

### Step 5. 실행

터미널에 **`hermes`** (또는 `HERMES`) 입력 → 노란색 에이전트 화면이 뜨면 **설치 성공**.

## 3. 자주 막히는 곳

| 증상 | 대응 |
|---|---|
| 파이썬 설치 오류 | 오류 화면을 클로드/챗GPT에 주고 "파이썬 설치 코드 줘" 요청 |
| 설치가 오래 걸림 | uv가 의존 패키지를 처음 까는 중 — 사양에 따라 다름, 대기 |
| 로그인 캡차 에러 | 브라우저 바꾸거나 새로 시도 |
| 설정을 잘못 선택함 | `hermes setup`으로 처음부터 재설정 (버셀 샌드박스 → 로컬 변경 가능) |
| 1번 로컬 vs 7번 로컬? | 차이 없음. 7번 = "이전 선택값 그대로" |
| sudo 켜야 하나? | **무조건 N** (관리자 권한 = 위험, 처음엔 끄기) |

> 설정 완료 화면을 캡처해 두면 오늘 설정을 언제든 재현 가능. 다시보기도 제공됨.
