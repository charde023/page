컨설팅 관점 분석 & 카피 지침

출처: 02_글쓰기/컨설팅.pdf — 외부 컨설턴트가 v12 디자인 시안(상품명 "에이폼 분수물총", 4세 기준)에 빨간펜으로 단 리뷰. 형식: 왼쪽 = 현재 우리 카피(취소선·X·밑줄·교정어), 오른쪽 = 컨설턴트가 다시 쓴 문장. 이 문서의 목적: 코멘트 하나하나가 왜 나왔는지 역추론하고, 컨설턴트가 일관되게 적용한 관점(원칙)을 지침으로 박제한다. 사장님이 이 문서를 보며 다시 질문하고, 그 관점을 체화하는 학습 교재로 쓴다. 작성: 2026-05-27. 분석자: Claude.

0. 한 줄 요약 — 컨설턴트가 일관되게 한 일

"정보를 전달하는 카피"를 "엄마에게 말을 거는 카피"로 바꿨다. 그 과정에서 ① 스펙을 뒤로 빼고 감정·체험을 앞세웠고, ② 거의 모든 기능을 '안전' 프레임으로 다시 포장했으며, ③ 아이의 '스스로'와 엄마의 '휴식'이라는 두 단어를 인과로 묶고, ④ 문체를 옆집 언니가 말해주는 부드러운 구어체(~, ..., !)로 녹였다."

엄마 타겟에서 카피의 적은 "재미없음"이 아니라 "불안 + 정보 과부하 + 남 얘기 같음"이다. 컨설턴트의 모든 칼질은 이 세 적을 겨눈다.

1. 컨설턴트의 13가지 핵심 관점 (지침)

각 항목: 원칙 → 근거가 된 실제 수정 → 왜(추론) → 우리 적용 지침.

원칙 1. 히어로에서 스펙을 들어낸다 (감정·체험 선두)

  • 수정: B1 바디 에이폼 분수물총 핑크+블루 두 자루 / 배터리도 충전도 없어, 두 줄을 취소선으로 삭제. 히어로에는 장면(미들헤드)만 남김.
  • 왜: 첫 화면에서 제품명·구성·배터리 같은 스펙을 던지면 "광고가 시작됐다"는 신호가 켜진다. 엄마가 사는 건 물총이 아니라 "내가 그늘에서 쉬는 장면"이다. 스펙은 B5·B12에서 충분히 나오니 히어로에서 중복으로 소모하지 않는다.
  • 지침: B1은 감정 장면 + 혜택 단어만. 제품명·구성·배터리·연령 같은 명세는 히어로에서 뺀다. (우리 글쓰기프로세스 B1 "감정 95% : 논리 5%"와 정확히 일치 — 우리가 시안에서 다시 스펙을 넣었던 걸 컨설턴트가 도로 빼준 것)

원칙 2. 모든 기능을 '안전' 프레임으로 다시 포장한다

  • 수정:
  • B2: 사방으로 동그랗게 퍼지는 분수예요 → 사방으로 동그랗게 퍼지는 안전한 분수물총
  • B5 카드2: 퍼지는 분수형이라 덜 아파요 → 앞쪽으로 뿜어내는 **위험한** 물총이 아니라 예쁘게 퍼지는 **안전한** 분수물총이에요
  • B6: 전동이 안 되는 단지 물놀이장 → 전동물총은 물길이 너무 세고 배터리 사고도 **위험해서** 전동물총 사용이 금지된 물놀이터가 생각보다 많아요!
  • B11 FAQ: 수동 펌프식이라 전동 워터건이 안 되는 곳 → **안전한 물총**이라 어느 장소에서도 사용 가능해요
  • 왜: 엄마의 1차 구매 장벽은 "재밌나?"가 아니라 "안전한가? 우리 애가 다치거나 남의 애를 다치게 하지 않나?". 컨설턴트는 분수형·수동펌프 같은 기능(Feature)을 전부 "그래서 안전하다"는 혜택(Benefit)으로 번역했다. 경쟁(전동물총)을 "위험한 것"으로 세워 우리를 "안전한 것"으로 대비시키는 구도도 만든다.
  • 지침: 기능을 쓸 때마다 "그래서 엄마가 안심할 수 있다"까지 끌고 간다. 단, AGENTS.md 금지선(과장된 안전 표현 금지)을 넘지 않게 — "절대 안 다쳐요" 같은 보장형은 금지. "덜 아파요 / 안심돼요 / 안전한 분수물총" 수준까지만.

원칙 3. '스스로 / 혼자'(아이 독립)를 엄마 휴식의 원인으로 명시한다

  • 수정:
  • B3 전환: 올해는 다를 거예요! → 이제는 분수물총으로 **아이 스스로 놀게** 해주세요!
  • B4: 아이가 혼자 물 채우고 → 아이가 **스스로** 물 채우고, 그리고 **스스로** 맘껏 놀고나면 집에 오자마자 곯아 떨어질거예요
  • 왜: "엄마가 쉰다"는 결과만 외치면 공허하다. 엄마는 "왜 내가 쉴 수 있는데?"의 답을 원한다. 답은 "아이가 스스로 노니까". 컨설턴트는 결과(쉼)와 원인(아이의 자립)을 한 문장 안에 묶어 약속을 믿게 만든다.
  • 지침: '엄마 쉼'을 말할 땐 항상 그 옆에 '아이가 스스로/혼자'라는 메커니즘을 붙인다. 이 둘은 분리하지 않는다.

원칙 4. 혜택을 '명사'로 못박는다 (휴식 / 쉼)

  • 수정:
  • B1: 엄마는 그늘에 잠깐 앉아 있어도 돼요 → 엄마는 그늘에서 **휴식**을~
  • B4: 그늘에 앉아 지켜보기만 해도 돼요 → 그늘에 앉아 **쉬셔도** 되요~
  • 왜: "앉아 있어도 돼요"는 행동 묘사일 뿐 혜택이 흐릿하다. "휴식 / 쉼"이라는 명사로 박으면 엄마가 사는 것의 정체가 또렷해진다. (이 프로젝트 약속 자체가 "엄마의 쉼"이므로 카피 표면에 그 단어가 반복돼야 한다.)
  • 지침: 혜택은 동작 서술("~해도 돼요")로 흘리지 말고, 핵심 명사(휴식·쉼)를 카피 표면에 직접 노출한다.

원칙 5. 정보체 → 옆집 언니 구어체 (의성어·물결표·말줄임)

  • 수정: 푹 → **푸욱~**, 밀면 → **쭈욱** 밀면, 문장 끝마다 ~ 대량 추가(노는 동안~, 휴식을~, 재미죠!), ... 추가(물총 맞느라 옷은 다 젖고...), 반짝 → **반짝반짝**.
  • 왜: 딱 떨어지는 정보체는 "광고/설명서" 톤이다. 늘어뜨린 의성어와 물결표는 말로 건네는 느낌을 만들어 경계심을 푼다. 우리 자가진단의 "영업맨 테스트(친구에게 말로 읽어 어색하지 않은가)"를 통과시키는 장치.
  • 지침: 감정 구간(B1~B4·B7)은 소리 내어 말하는 리듬으로. 의성어 늘이기(푸욱~), 물결표, 말줄임표를 톤 메이커로 쓴다. 단 마침표는 여전히 생략(우리 문체 규칙 유지). 주의: 논리 구간(B11·B12)까지 물결표를 남발하지 않는다 — 거기선 신뢰가 톤이다.

원칙 6. 브랜드 놀이명을 일관되게 + 단어를 '예쁘게' 고른다

  • 수정: 물꽃에 → **물꽃놀이에** (브랜드 경험명 통일), 물방울이 → **물빛**이, 반짝 → 반짝반짝.
  • 왜: 제품을 "물총"이 아니라 '물꽃놀이'라는 고유 경험으로 계속 부르면 카테고리가 우리 것이 된다(B2의 정체성 목표). "물방울"보다 "물빛"이 시각적으로 더 곱고 사진/햇빛 맥락과 어울린다 — 엄마가 상상하는 그림을 더 예쁘게.
  • 지침: 경험은 항상 '물꽃놀이'로 호명. 감각 묘사어는 더 곱고 시각적인 쪽으로 고른다(물방울→물빛, 반짝→반짝반짝).

원칙 7. 군더더기 수식어를 쳐낸다 (헤드는 짧을수록 세다)

  • 수정: B1 헤드 엄마를 **한참** 안 찾아요에서 한참을 X로 삭제 → 엄마를 안 찾아요. (앞서 사장님도 B1 미들 잠깐을 삭제한 것과 같은 결)
  • 왜: "한참"은 강도를 재는 부사라 오히려 약속을 흐린다("한참=얼마나?"). 빼면 엄마를 안 찾아요가 더 단정적이고 강하다. 헤드라인은 덜어낼수록 후크가 선명해진다.
  • 지침: 헤드·핵심 약속 문장에서 강도 부사(한참·잠깐·정말·매우)는 의심하고, 빼도 뜻이 사는지 본다. 살면 뺀다.

원칙 8. 연령 수치 주장을 능력 표현으로 바꾼다 (표시광고 리스크 회피 + 타겟 확대)

  • 수정:
  • B4: 4세도 혼자 채워요 → 아이도 쉽게 할 수 있어요
  • B11 FAQ: 4세 이상 권장이에요 / 4세도 손잡이를 혼자 당겨요(밑줄로 표시) → **혼자 당길 수 있어요 / 혼자 할 수 있어요**
  • B1 바디의 4세부터 혼자도 빨간색으로 플래그(재검토 표시)
  • 왜: "4세"는 ① KC 실제 권장연령과 어긋날 위험(표시광고), ② 4세 못 미치는 아이 부모를 배제. "혼자 할 수 있어요"는 숫자 주장을 피하면서 핵심 혜택(자립)은 그대로 전한다. AGENTS.md "권장연령은 KC 문서와 일치 확인 전까지 확정 표현 주의"와 정확히 같은 방향.
  • 지침: 연령은 숫자 단정 대신 능력 서술("혼자 당길 수 있어요")로. 숫자는 KC 인증서 확정 후에만, 그것도 스펙표(B12)에서만.

원칙 9. 효과는 단정하지 말고 추정형으로 (보장형 회피)

  • 수정: B4 곯아떨어져요(단정) → 곯아 **떨어질거예요**(추정). 부르는 일이 줄어요 → 부르는 일이 **없을거예요**도 추정 톤.
  • 왜: "곯아떨어짐"은 자생리뷰 근거가 없는 비트(우리 v1 주석에 명시됨). 단정하면 효과 보장이 되어 위험. ~거예요 추정형은 "흔한 저녁 풍경"으로 그리되 보장은 피하는 안전한 절충.
  • 지침: 근거 없는 결과(꿀잠·곯아떨어짐 등)는 추정형(~거예요/~겠죠)으로만. 단정·수치·"매일·반드시"는 금지(AGENTS.md).

원칙 10. 1+1을 '혜택·선물'로 적극 표현하고 실용 이유를 붙인다

  • 수정:
  • B5 카드: 두 자루가 와요 → **1+1** 두 자루를 **드려요!** + 니꺼 내꺼 구분하기 쉽게 핑크 + 블루 한세트를 보내드려요
  • B8: 한 번에 둘이에요 → **1+1**, 둘이 함께 놀 때도 앞에 **물놀이는 같이해야 재미죠!** 추가, 한 자루는 집에 → 한 자루는 **예비용으로~**
  • 왜: "두 자루가 와요"는 사실 진술. "1+1 드려요!"는 받는 사람을 이득 본 기분으로 만든다(증정 프레이밍). 그리고 왜 2개가 좋은지(같이 놀아서 재밌다 / 하나는 예비) 실생활 이유를 줘야 가성비 논리가 산다.
  • 지침: 구성은 사실 나열이 아니라 "드려요/보내드려요" 증정 톤 + 2개라서 좋은 실생활 이유로. 단 AGENTS 금지 프레임("내 것 명확/형제 싸움")은 피하고 "구분하기 쉽게/예비용"처럼 중립적 이유로.

원칙 11. 감정 마무리는 행동 유도(명령형)로 전환한다

  • 수정: B3 전환 올해는 다를 거예요!(감정 선언) → 이제는 분수물총으로 아이 스스로 **놀게 해주세요!**(행동 권유)
  • 왜: 감정 선언은 여운은 있지만 독자를 움직이지 않는다. "~해주세요"는 다음 행동(계속 읽기/구매 고려)으로 미는 작은 CTA. 본문 곳곳에 미니 CTA를 심어 미끄럼틀 속도를 올린다.
  • 지침: 섹션 전환에서 가끔 감정 선언을 행동 권유형으로 바꿔 추진력을 준다(남발은 금지, 핵심 전환에만).

원칙 12. 전문어·기술어를 엄마 일상어로 바꾼다

  • 수정: B2 직선으로 쏘는 물총이 아니라 → **앞쪽으로** 쏘는 물총이 아니에요. 물이 빙 둘러 퍼지면서 → **물길이 분수처럼** 퍼지면서.
  • 왜: "직선"은 기하/기술 용어라 차갑고, 동시에 위험의 방향(앞=얼굴 쪽)을 못 짚는다. "앞쪽으로 쏘는"은 일상어이면서 "남의 애 얼굴로 안 간다"는 안전 함의까지 깐다(원칙 2와 결합).
  • 지침: 기술어(직선/방사형/회오리)는 엄마가 입으로 쓰는 말로 번역하되, 가능하면 안전·편익 함의를 담아서.

원칙 13. 카피라이터가 레이아웃까지 본다 (중앙정렬·불필요한 전환 삭제)

  • 수정: B5·B6 오른쪽에 ←중앙정렬 지시. B2→B3 전환이던 그런데! 엄마들이 더 좋아해요에 큰 X → 삭제.
  • 왜: 카피의 효과는 배치에서 완성된다. 카드 설명은 중앙정렬이 시선이 안정적이고, 군더더기 피벗 라인은 미끄럼틀을 끊으니 삭제. 컨설턴트는 글만 보지 않고 "읽히는 그림" 전체를 본다.
  • 지침: 카피를 넘길 때 정렬·삭제 가능한 전환 라인까지 함께 제안한다. 모든 피벗이 필요한 건 아니다 — 흐름을 끊는 전환은 과감히 뺀다.

2. 카피별 한땀한땀 (Before → After)

표기: [원본] = 시안(왼쪽) / [컨설턴트] = 리라이트(오른쪽) / 의도 = 추론한 이유. ⚠ 컨설턴트 원문에는 맞춤법 슬립(되요→돼요, 너무쎄고→너무 세고, 니까→내꺼)이 섞여 있다. 의도·톤만 흡수하고 표기는 우리 맞춤법으로 정제한다.

B1 — 히어로

3. 용어 사전 — 단어 하나하나의 고민

바꾼 것 방향 한 줄 이유
물꽃 → 물꽃놀이 제품 → 경험 카테고리를 우리 고유 경험으로 호명
물방울 → 물빛 평범 → 예쁨 햇빛·사진 맥락에 더 곱고 시각적
직선으로 → 앞쪽으로 기술어 → 일상어+안전 차갑지 않고, 위험 방향(얼굴)을 함의

4. 우리 기존 지침과의 정합 / 충돌 체크

  • 정합(컨설턴트가 우리 원칙을 복원해 줌):
  • 원칙1(히어로 스펙 제거) = 글쓰기프로세스 B1 "감정 95:논리 5". → 우리가 시안에서 다시 넣은 스펙을 도로 빼준 것.
  • 원칙8(연령 능력화) = AGENTS.md "권장연령 확정 표현 주의".
  • 원칙9(추정형) = AGENTS.md "효과 보장·미검증 수치 금지", v1 곯아떨어짐 주석.
  • 주의(컨설턴트 제안을 그대로 쓰면 우리 금지선에 닿을 수 있음):
  • 원칙2 안전 강조는 좋되, "절대/100% 안전" 같은 과장 안전 표현 금지(AGENTS) 선을 넘지 말 것. "안전한 분수물총/덜 아파요"까지만.
  • 원칙10 1+1 이유에서 "내꺼 구분"은 우리 금지 프레임("내 것 명확/형제 싸움")과 닿는다. → "색이 둘이라 섞이지 않아요 / 예비용으로" 같은 중립 표현으로 정제해 수용.
  • 컨설턴트 원문 맞춤법 슬립(되요/돼요, 너무쎄고)은 반드시 교정. 우리 문체(마침표 생략·해요체)는 유지.
  • 확인 필요(시안 vs v2 불일치): 컨설팅 대상 시안은 상품명 "에이폼 분수물총", 연령 4세 기준. v2 카피본은 "펌프식 회오리 분수물총"·5세. → 현재 SSOT는 v6+ 이력(에이폼·4세)이 맞는지, KC 인증 연령과 4세가 일치하는지 발행 전 게이트로 재확인.
