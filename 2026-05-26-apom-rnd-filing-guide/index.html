1신고 절차 (선설립 후신고)

연구소를 실제로 갖춘 뒤 KOITA(rnd.or.kr)에 온라인으로만 신고. 수수료 무료, 법정 처리 10일.

순서단계핵심
1사전 준비인사발령·독립공간·기자재·현판 (신고일 이전 완료)
2공인인증서사업자번호(본점) 기업용 인증서
3자가진단rnd.or.kr 체크리스트로 적격 점검
4온라인 신청기업→연구소→활동개요서→인력→기자재 입력
5첨부 업로드조직도·도면·사진·증명서 (모두 JPG)
6심사·보완보완요청 시 기한 내 제출 (초과 시 반려)
7인정·출력인정서 온라인 출력

제출서류 13종: 신고서·활동개요서·사업자등록증·조직도·도면·사진·기자재현황·인사발령장·4대보험명부·학위/자격증·경력증명서·업무분장표·중소기업확인서.

2하드웨어 — 독립 연구공간

핵심 원칙

사방 고정 벽체 + 별도 출입문. 천장 시공 불가 시 바닥에서 2m 이상 벽체. 이동식 파티션만으론 불인정(50㎡ 이하 예외 제외). 유리벽도 사방 막히면 인정.

50㎡ 이하 예외

중소·벤처기업이 전용면적 50㎡ 이하면 파티션·책장·이격 구분도 인정 → 신고서 "분리구역". 단 50㎡ 초과 확보 불가한 경우만.

  • 에이폼 권장: 사무동 2층 = 연구소 본공간. 33평은 50㎡ 초과 가능성 커서 고정벽체+출입문 독립공간 구성이 안전
  • 실험실과 연구원 사무공간 둘 다 독립이어야 함 (실험실만 분리하면 불인정)
  • 현판: "(주)에이폼 ○○연구소" — 인정서 명칭과 일치, 반영구 재질, 연구소 출입문에 부착
  • 도면: 층 전체도면 + 내부도면(치수·면적계산식·좌석·기자재·출입문·현판 위치), A4, JPG

3연구기자재 구비

✅ 인정

연구전용 PC·SW(CAD·시뮬·분석), 시험·측정 장비, 3D프린터·시제품 장비, 실험장비 — 연구소 내부에 전용 배치.

❌ 불인정

사무용 PC·복합기·전화, 서버·허브, 범용 측정기(버니어캘리퍼스·전자저울), 타 부서 공동사용 장비, 연구소 밖 장비.

  • 최소 금액·수량 법정기준 없음. 단 장비 거의 없는 회의실 형태는 불인정
  • 기자재 현황표(별지3호): 연번/품명/모델/수량/제작사/도입일/취득금액/설치장소/사용구분 — 고가순, 엑셀 업로드
  • 에이폼 예시: (제조) 두께측정기·낙하충격 지그·인장압축시험기·시제품 금형 / (물류) 연구전용 PC+개발·분석 SW

5인사발령·증빙 양식

  • 인사발령장: 발령일자를 설립신고일 이전으로. "○○연구소 연구전담요원(전임) 발령". 회사 자체 양식
  • 업무분장표: 담당업무란에 연구개발만 기재(생산·영업·품질·물류 금지) → 겸직 배제 입증
  • 자격 증빙: 학위증명서/자격증, (경력자)경력증명서+국민연금 가입내역(기간 일치 필수), 4대보험 가입자명부
  • 연구소장: 전임=인원 포함·겸직 불가 / 겸임=인원 불포함

🔴 대표님 연구전담요원 — 3년 시한

대표이사는 창업 3년 이내 소기업일 때만 인정. 개업 2024-03-01 → 2027-03-01 이후 변경신고로 제외 필수. 제외 시 정규직 1명만 남아 소기업 기준(3명) 미달.

대응: 2027-03 전 연구원 1명 추가 채용 또는 벤처확인(연구개발유형) 취득. 대표님 인건비는 Time Sheet·연구노트 없으면 세액공제 전액 부인.

6활동개요서·과제계획서·연구노트

연구개발활동개요서

신고용. 전문연구분야를 과학기술 분류로("제조업 전반/IT"❌ → "화학공학·고분자/컴퓨터·SW·산업공학"✅). 분야 2~3개 집중, 기대효과에 기술적 성과 포함.

과제계획서(별지3호의2)

세무 세액공제·보관용(과제별, 5년). 과제명·기간·예산·목표·방법·기대성과·월별 WBS·참여율. 양산·QC는 과제 불가.

  • 연구노트: 활동 당일 기록(소급 금지), 실패·오류수정 과정이 핵심 증거, 겸직자는 Time Sheet 병행
  • 에이폼 권장 과제: "HDPE 블로우 성형 공정조건 최적화", "전자상거래 데이터 기반 수요예측+피킹 동선 WMS 모듈"

7혜택 & 사후관리

  • 법인세 R&D 세액공제: 중소기업 당기분 25%/증가분 50%. 별지1호 신청서+별지3호 명세서+총괄표. 국세청 사전심사 제도 권장
  • 지방세 감면: 비수도권(경산) 연구소용 부동산 취득세·재산세 감면 → 경산시 세무부서 확인. 4년 내 폐쇄·용도변경 시 추징
  • 변경신고: 연구원·소재지 변경 시 발생일로부터 14일 이내. 연구개발활동조사: 매년 4월 온라인 제출

8핵심 주의사항

함정결과 → 회피
연구원이 생산·영업 겸직(업무분장표 기재)자격·공제 부정 → 연구업무만 기재
경력증명서 ≠ 국민연금 기간경력 전체 불인정 → 사전 대조
실험실만 독립, 좌석은 공용전면 불인정 → 둘 다 독립
장비를 타 부서와 공동사용기자재 불인정 → 연구전용
활동개요서가 산업분류·광범위반려 → 과학기술 분야 2~3개 집중
인사발령일이 설립일보다 늦음보완/불인정 → 설립일 이전
보완요청 기한 초과반려(재신고) → 담당자 수시 확인
변경신고·활동조사 미이행인정 취소·추징 → 기한 준수
